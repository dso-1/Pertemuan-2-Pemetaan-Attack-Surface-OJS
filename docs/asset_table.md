# Klasifikasi Aset Kritis (CIA Triad)

Daftar aset utama sistem OJS yang diklasifikasikan berdasarkan tingkat kerahasiaan, integritas, dan ketersediaannya.

| Aset | Confidentiality | Integrity | Availability | Nilai Kritis |
| :--- | :--- | :--- | :--- | :--- |
| **Data login admin** | Tinggi | Tinggi | Sedang | **Kritis** |
| **Naskah belum terbit** | Tinggi | Tinggi | Sedang | **Kritis** |
| **File konfigurasi (config.inc.php)** | Tinggi | Tinggi | Sedang | **Kritis** |
| **Kredensial database** | Tinggi | Tinggi | Rendah | **Kritis** |
| **Data reviewer** | Sedang | Tinggi | Sedang | **Tinggi** |
| **Artikel terbit (published)** | Rendah | Tinggi | Tinggi | **Tinggi** |
| **Log file server** | Sedang | Sedang | Sedang | **Sedang** |
